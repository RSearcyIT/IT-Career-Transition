# OSI Model (7 Layers)

The OSI (Open Systems Interconnection) Model is a framework used to understand how data travels across a network.

## Layer 7 - Application
Provides network services directly to users.

Examples:

- HTTP
  
- HTTPS
  
- FTP
  
- DNS

## Layer 6 - Presentation
Handles data formatting, encryption, and compression.

Examples:

- SSL/TLS
  
- JPEG
  
- MP3

## Layer 5 - Session
Manages communication sessions between devices.

Examples:

- NetBIOS
  
- RPC

## Layer 4 - Transport
Provides reliable data delivery.

Protocols:

- TCP
  
- UDP

## Layer 3 - Network
Responsible for routing packets.

Examples:

- IP Addresses
  
- Routers

## Layer 2 - Data Link
Handles communication between devices on the same network.

Examples:

- Switches
  
- MAC Addresses

## Layer 1 - Physical
Physical transmission of data.

Exwmples:

- Ethernet Cables
  
- Fiber Optics
  
- Wireless Signals
