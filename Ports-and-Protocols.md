# Ports and Protocols

## What is a Port?

A port is a communication endpoint used by applications and services on a network.

Example:
An IP address identifies a device.
A port identifies a specific service on that device.

## Common Ports

| Protocol | Port | Purpose |
|----------|------|---------|
| HTTP | 80 | Web traffic |
| HTTPS | 443 | Secure web traffic |
| DNS | 53 | Name resolution |
| SSH | 22 | Secure remote access |
| FTP | 21 | File transfers |
| RDP | 3389 | Remote Desktop |

## Why Ports Matter

Ports allow multiple services to run on the same device while keeping network traffic organized.

## Interview Question

What port does HTTPS use?

HTTPS uses port 443.
