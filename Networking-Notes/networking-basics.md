# Networking Basics

## What is a Network?

A network is a group of devices connected together so they can communicate and share resources.

Examples:

- Computers

- Servers

- Printers

- Smartphones

## IP Address
An IP address identifies a device on a network
Example:
192.168.1.100
## Switch
A switch connects devices within the same local network
## DNS
DNS (Domain Name System) translates website names into IP addresses
Example:
google.com-> IP address

## DHCP
DHCP (Dynamic Host Configuration Protocol) automatically assigns IP addresses to devices on a network
Example:
When a laptop connects to Wi-Fi, DHCP assigns it an IP address automatically
## TCP/IP
TCP/IP is the communication protocol used by devices on the internet and local networks
TCP = reliable data delivery
IP = addressing and routing
## Routing
Routers connect different networks together and direct traffic between them
Example:
A home router connects your local network to the internet
