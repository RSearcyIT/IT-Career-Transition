# Networking Basics
## What is a Network?
A network is a group of devices connected together so they can communicate and share resources.
Examples:
Computers
Servers
Printers
Smartphones
## IP Address
An IP address identifies a device on a network
Example:
192.168.1.100
## Switch
A switch connects devices within the same local network
## DNS
DNS (Domain Name System) translates website names into IP addresses
Example:
google.com-> IP address
